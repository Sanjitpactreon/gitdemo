Commands for GitÔÉÅ

These are basic commands for git

  1. init: This command is used to initialise git in a directory

git init

  1. add: This command adds the new files to the staging

git add

  1. commit: this is used to commit the added files

git commit -m "comitting"

  1. push: this pushes the committed files to the remote directory

git push -u origin master

  1. status: this gives the current status of the directory

git status

  1. pull: This command is used to pull from a remote directory

git pull

  1. branch: This command is used to create a new branch

git branch -M <branch name>

  1. merge: this is used to merge the current branch to the main branch

git merge <branch name>
